# calculator_main

This is my final project for The Odin Project Foundations course.

Features:
- Basic math operations: add, subtract, multiply, divide
- Operate function to handle calculations
- Clear (C) and All Clear (AC) buttons
- Backspace (DEL) support
- Division by zero handled with a snarky error
- Consecutive operator handling
- Rounded decimal results
- Number formatting with commas
- Scrollable display for long numbers
